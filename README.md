---

### 🛠 現在の学習状況
実務未経験の技術に対し、ドットインストールやProgate等のプラットフォーム、および自習用スクリプトを通じて、以下の技術を習得中です。

- **Python (Main Focus)**
  - 基礎文法、反復処理、辞書・リスト操作、ファイル操作
  - **こだわり:** 単なる写経ではなく、「例外処理」や「無限ループ防止」などの運用視点を意識したコメント・アレンジを加えています。
- **JavaScript (ES6+)**
  - フロントエンドの基本操作、DOM操作、非同期処理の基礎。
- **PHP**
  - サーバーサイドの基本構造とWebシステムの裏側の理解。

---

### 📂 リポジトリの構成
- `01_Basic_Syntax.py`: 変数、条件分岐、データ型の基礎
- `02_Loops_and_Collections.py`: for/while文、辞書・リストの応用、添字演算子の理解
- `JS_Basic_Practice.js`: JavaScriptの基本構文
