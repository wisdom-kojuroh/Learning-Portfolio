なぜ「0」と「1」だけなのか？

コンピュータは電気回路の集合体です。そして電気には「電圧が高いか、低いか」の2つの状態しかありません。

コンピューターは「1」と「0」の世界。という言葉を聞いたことはないですか？

それはつまり、電圧の高いか低いかを「1」と「0」に対応させているからです。

2進数： 電気が「ON/OFF」の2種類だから採用された（0〜1の2種類）。2. 「ビット（bit）」と「バイト（byte）」bit（ビット）： 0か1が入る最小の箱のこと。

byte（バイト）： 8ビット集まったもの。なぜ8個か？それは「英数字1文字」を表現するのにちょうどいいサイズだったからだ。

2進数の数え方（ここが試験に出る！）

右から順に、1の位、2の位、4の位、8の位…と「2倍」ずつ増えていく。2進数の「1010」は10進数でいくつ？(8の位×1) + (4の位×0) + (2の位×1) + (1の位×0) = $8 + 0 + 2 + 0 = 10$ だ！
